# 实时性与定时器

## 面试高频问答总结

### 1. 实时性在内核中的体现？
- 任务按时完成，响应延迟可控。

### 2. 常见定时器类型及区别？
| 类型      | 精度     | 适用场景         |
| --------- | -------- | ---------------- |
| 内核定时器| 毫秒级   | 普通定时任务     |
| hrtimer   | 微秒级   | 高精度定时       |
| tasklet   | 软中断下 | 轻量级延迟任务   |

#### 定时器工作流程图
```mermaid
flowchart LR
    A[设置定时器] --> B[等待超时]
    B --> C[触发回调/任务]
```

### 3. 实时调度策略有哪些？
- RT内核、优先级调度、抢占式调度。

### 4. 形象比喻
- 实时性像“闹钟准时响”，定时器像“定时提醒”。

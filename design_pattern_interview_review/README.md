# 设计模式面试复习（由浅入深）

> 用法：
> 1. 先只看题目进行自测；
> 2. 再跳转到答案核对；
> 3. 遇到关联模式，继续点击链接横向复习。

## 建议复习节奏

- 第 1 轮：只看各章 `questions.md`，按题号口述。
- 第 2 轮：对照各章 `answers.md`，补知识盲区。
- 第 3 轮：沿着“关联模式”链接做对比（例如策略 vs 状态、代理 vs 装饰器）。

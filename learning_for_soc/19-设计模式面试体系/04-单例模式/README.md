## 一句话定义
确保类只有一个实例，并提供全局访问点。

## 关键知识点
- C++11 局部静态变量初始化具备线程安全保证。
- 删除拷贝构造与赋值运算，防止复制实例。
- 谨慎使用：全局状态会提高耦合、降低可测试性。

## 图解（Mermaid）
```mermaid
flowchart TD
A[getInstance()] --> B{实例已创建?}
B -- 否 --> C[创建唯一实例]
B -- 是 --> D[返回现有实例]
C --> D
```

## 面试答题模板（30~60秒）
1. 先说定义：确保类只有一个实例，并提供全局访问点。
2. 再说适用场景与优缺点。
3. 最后给一个 C++ 落地点（接口抽象、智能指针、生命周期管理）。
